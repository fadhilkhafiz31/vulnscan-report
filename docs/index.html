A lightweight Nmap-based vulnerability scanner

vulnscan-report is a Python-based security tool that automates Nmap scans and generates clean HTML vulnerability reports. Perfect for cybersecurity professionals and security enthusiasts.

View Sample Reports View GitHub Repository

üìñ Usage Examples

Default Scan:
py run_scan.py --target 127.0.0.1 --output reports/my_report.html
Fast Scan:
py run_scan.py --target 127.0.0.1 --profile fast --output reports/quick_scan.html
Full Port Scan:
py run_scan.py --target 192.168.1.1 --profile full --output reports/full_scan.html
Top 100 Ports:
py run_scan.py --target 10.0.0.1 --profile top100 --output reports/top100_scan.html
UDP Scan:
py run_scan.py --target localhost --profile udp --output reports/udp_scan.html

Note: All reports use the modern dark mode interface with severity scoring, color-coded rows, and professional styling.
